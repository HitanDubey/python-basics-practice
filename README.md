# Python Practice Repository

A comprehensive collection of Python practice programs covering basics, intermediate concepts, Data Structures & Algorithms (DSA), and Object-Oriented Programming (OOPs).

## Repository Structure
├── basics/ # Basic Python programs
├── intermediate/ # Intermediate concepts
├── dsa/ # Data Structures & Algorithms implementations
└── oops/ # Object-Oriented Programming examples


## Topics Covered

### Basics
- String manipulation & slicing
- Loops & conditionals
- Functions
- Number operations (Armstrong, Fibonacci, Prime, GCD, LCM)
- Frequency counters
- Palindrome checking
- Factorial (iterative & recursive)

### Intermediate
- Pattern printing (hollow triangles, right-angle triangles)
- Games (Tic-Tac-Toe with Minimax AI, Guess the Number)
- Advanced string operations
- Prime factorization
- Next prime number
- Variable-length arguments

### DSA
- Data Structures implementations
- Algorithms practice
- *(Contents will be added)*

### OOPs
- Classes and objects
- Inheritance (single inheritance)
- Encapsulation
- Polymorphism
- Dog class example
